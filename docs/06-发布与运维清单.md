# 06-发布与运维清单

## 1. 发布前清单

- 协议版本已声明
- 设备 Manifest 已填写
- 基础测试全部通过
- 压测与兼容测试通过
- 安全策略启用
- 文档与示例同步更新

## 2. GitHub 最低配置

- README
- LICENSE
- CHANGELOG
- CONTRIBUTING
- SECURITY
- ISSUE / PR 模板

## 3. 运维建议

- 记录连接成功率
- 记录平均发送耗时
- 记录断连频次
- 记录错误码分布

## 4. 版本升级策略

- 小版本：修复与优化，不破坏兼容
- 中版本：新增能力，默认向后兼容
- 大版本：协议破坏性升级，需迁移说明

## 5. 社区支持建议

- FAQ 常见故障库
- 设备适配案例库
- 每月兼容设备更新公告

## 6. 风险应对

- 发现安全漏洞：先发布风险公告，再给补丁版本
- 出现严重断连：提供临时降级方案
- 社区出现魔改固件：明确“官方兼容清单”边界
